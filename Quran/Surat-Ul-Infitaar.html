Surah Infitaar
﴾o﴿بِسۡمِ اللّٰہِ الرَّحۡمٰنِ الرَّحِیۡمِ
شروع کرتا ہوں اللہ تعا لٰی کے نام سے جو بڑا مہربان نہایت رحم والا ہے ۔
﴾۱﴿اِذَا السَّمَآءُ انۡفَطَرَتۡ ۙ
جب آسمان پھٹ جائے گا ۔
﴾۲﴿وَ اِذَا الۡکَوَاکِبُ انۡتَثَرَتۡ
اورجب ستارے جھڑ جائیں گے ۔
﴾۳﴿وَ اِذَا الۡبِحَارُ فُجِّرَتۡ
اور جب سمندر بہہ نکلیں گے ۔
﴾۴﴿وَ اِذَا الۡقُبُوۡرُ بُعۡثِرَتۡ
اور جب قبریں ( شق کر کے ) اکھاڑ دی جائیں گی ۔
﴾۵﴿عَلِمَتۡ نَفۡسٌ مَّا قَدَّمَتۡ وَ اَخَّرَتۡ
۔ ( اس وقت ) ہر شخص اپنے آگے بھیجے ہوئے اور پیچھے چھوڑے ہوئے ( یعنی اگلے پچھلے اعمال ) کو معلوم کر لے گا ۔
